## Grocery Price Comparison Analysis (Pak’nSave vs Woolworths)

### Project Overview
This project is an end-to-end grocery price comparison analysis completed entirely in **Microsoft Excel**, using real-world data collected from personal weekly grocery shopping over an eight-week period.

The goal was to objectively compare grocery prices across retailers, identify pricing patterns, and understand potential savings while avoiding bias in the analysis.

---

### Data Collection
- Collected data from **8–10 weekly grocery receipts** over approximately **2 months**
- Included weekly specials and deals
- Compared **50+ grocery items** to reduce bias
- Ensured fair comparison by matching **same product type, quantity, and category**, even when brands differed
- Final dataset contained approximately **200 rows**

---

### Data Preparation & Structuring
- Entered raw data into structured **Excel tables**
- Created additional analytical columns based on project requirements
- Built a **master sheet** to consolidate and organise all data
- Standardised product names and categories to ensure consistency
- Connected tables using **VLOOKUP**

---

### Analysis Techniques Used
- Excel formulas and functions:
  - VLOOKUP
  - IF / IFS
  - AVERAGE
  - Basic arithmetic (add, subtract, totals)
- Pivot tables for:
  - Product-level comparisons
  - Weekly and category-based aggregation
- Conditional formatting to:
  - Highlight pricing differences
  - Identify items cheaper at Woolworths to avoid analysis bias

---

### Dashboard & Visual Analysis
Created an interactive Excel dashboard using:
- Charts and slicers for filtering
- Weekly grocery spend tracking
- Category-level spend analysis
- Product-level price comparison
- **8-week savings and profit rate analysis**
- Visual comparison to identify the cheaper retailer per product

---

### Key Insights
- **Pak’nSave was the cheaper retailer for 47 out of 50+ items**
- Achieved an overall **21.64% savings over 8 weeks**
- A small number of items were consistently cheaper at Woolworths, which were clearly highlighted to maintain objectivity
- Visual analysis showed clear trends in weekly spending and category-level costs

---

### Presentation & Outcome
- Findings were summarised into **PowerPoint slides**
- Presented insights to **mentors, teammates, and stakeholders** during **Demo Day**
- The project was **awarded 2nd place** for analysis quality, dashboard design, and presentation clarity
